# Here is a practice of fullstack development building a todolist app with ASP.NET:

## Stack:

|Layer|Technology|
| :--- | ---: | 
|Backend|ASP.NET Web API (C#)|
|ORM|Entity Framework Core |
|Database|SQLite(dev)|
|Frontend|React+Vite|
|Auth|ASP.NET|
|API Docs|Swagger (Swashbuckle)|
|Conteinerization|Docker+DockerCompose|
